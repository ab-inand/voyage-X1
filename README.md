A modern, AI-powered travel platform built with Next.js, React Three Fiber, and Tailwind CSS.

## Features

- 🌍 Interactive 3D Globe
- 🤖 AI Travel Concierge
- 🎫 Smart Booking System
- 🌟 Space Tourism
- 🚄 Hyperloop Integration
- 🏨 Underwater Hotel Experiences
- 🎯 Future Traveler Quiz
- 📧 Web3 Newsletter

## Tech Stack

- Next.js 14
- React Three Fiber
- Framer Motion
- Tailwind CSS
- TypeScript
- Three.js
